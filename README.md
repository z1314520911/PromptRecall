你好啊 

这是一款本地的网页应用，手机端也可访问。

通过提示/线索来帮助记忆和回忆简答题，

有三个锦囊提示，随着锦囊的增加，提示内容逐渐增加，也有完整的答案显示功能，

也有错题库，通过标记错题后上面会有错题列表，点击错误题号直接跳转对应题目。

并且有两种配色，男生版蓝色为主，女生版粉色为主。

通过上传excel文件来训练自己的记忆力

题库上传格式为两列，第一列是问题，以？结束，第二列是答案
